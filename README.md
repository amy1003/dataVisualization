# dataVisualization
一个使用djongo开发的数据可视化系统，前段使用了echart作为可视化工具,用reactJS包装了一层

# functions

1.上传保管数据

2.创建项目进行对数据可视化，可使用echart的一部分功能，如饼图，线图，柱状图，雷达图，地图，散点图等等....

3.实施跟踪数据:可以创建应用并运行，当后台数据改变，前台的可视化图表会同时变动，以达到监控数据的效果
